# product.distributed.consistenthashing
一致性哈希（Consistent Hashing）简单实现

# 功能简介

0、实现基于MurMurHash算法的一致性哈希实现

1、支持动态添加、移除node节点

2、支持node节点权重、虚拟节点、资源管理功能

3、支持redis类型node节点
